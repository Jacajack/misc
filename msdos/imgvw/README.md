# imgvw

An MS-DOS utility to view BMP3 images (up to 320x200 pixels).

Usage:
`imgvw [OPTIONS] FILE`

Supported options:
 - `-h` - display help message
 - `-g` - display image in grayscale
 - `-256` - display image in 256 colors
 - `-l <value>` - lightness (only in 256 color mode)
